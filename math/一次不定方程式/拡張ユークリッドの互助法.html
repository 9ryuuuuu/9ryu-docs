拡張ユークリッドの互除法は、通常のユークリッドの互除法で gcd⁡(a,b)\gcd(a, b)gcd(a,b) を求める過程に加えて、以下の等式

ax0+by0=gcd⁡(a,b)ax_0 + by_0 = \gcd(a, b)ax0​+by0​=gcd(a,b)

1. ユークリッドの互除法の復習

ユークリッドの互除法は、2つの整数 aaa と bbb の最大公約数を求めるアルゴリズムです。基本的な考え方は、次の性質に基づいています。

gcd⁡(a,b)=gcd⁡(b,a mod b)\gcd(a, b) = \gcd(b, a \bmod b)gcd(a,b)=gcd(b,amodb)

つまり、aaa を bbb で割った余り r=a mod br = a \bmod br=amodb に対して、gcd⁡(a,b)\gcd(a, b)gcd(a,b) は gcd⁡(b,r)\gcd(b, r)gcd(b,r) と同じになります。これを r=0r = 0r=0 になるまで繰り返します。

2. 拡張ユークリッドの互除法の目的

拡張ユークリッドの互除法とは、通常のユークリッドの互除法（最大公約数を求めるアルゴリズム）の発展形です。通常のユークリッドの互除法では、2つの整数 a と b の最大公約数 (gcd) を求めますが、拡張ユークリッドの互除法では、それに加えて、以下の形の線形結合

ax0+by0=gcd⁡(a,b)a x_0 + b y_0 = \gcd(a,b)ax0​+by0​=gcd(a,b)

を満たす整数 x0x_0x0​ と y0y_0y0​ も同時に求めることができます。

具体的には

  • 目的
    単に gcd(a,b)gcd(a, b)gcd(a,b) を計算するだけでなく、どのように aaa と bbb の組み合わせ（つまり、aaa と bbb の適切な倍数の和）でその最大公約数が表せるかを求めることが目的です。

  • なぜ重要か
    この線形結合の形は、後に一次不定方程式（例えば ax+by=cax+by=cax+by=c の解法）や、RSA暗号のような暗号アルゴリズムで秘密鍵を求めるときなど、さまざまな応用で利用されます。

  • まとめ

    拡張ユークリッドの互除法は、「aaa と bbb の最大公約数がどのような整数の線形結合で表せるか」という情報を同時に提供するため、単に gcdgcdgcd を求めるだけの通常のユークリッドの互除法よりも強力で、多くの実用的な問題（整数論、暗号、Diophantine 方程式など）の解法に応用される重要な手法なのです。

4. アルゴリズムの擬似コード

以下は、拡張ユークリッドの互除法の擬似コードです。

function extended_gcd(a, b):
    if b == 0:
        return (a, 1, 0)  // gcd(a,0)=a, 係数 x=1, y=0
    else:
        (d, x1, y1) = extended_gcd(b, a mod b)
        x = y1
        y = x1 - floor(a/b) * y1
        return (d, x, y)
  • 説明：
    • b=0b = 0b=0 の場合、直接 (a,1,0)(a, 1, 0)(a,1,0) を返します。
    • b≠0b \neq 0b=0 の場合、amod  ba \mod bamodb を用いて再帰的に gcd⁡(b,a mod b)\gcd(b, a \bmod b)gcd(b,amodb) と係数 x1,y1x_1, y_1x1​,y1​ を求めます。
    • 商 q=⌊a/b⌋q = \lfloor a/b \rfloorq=⌊a/b⌋ を使って、新しい係数 x,yx, yx,y を
      x=y1,y=x1−q y1x = y_1,\quad y = x_1 - q\,y_1x=y1​,y=x1​−qy1​
      として計算し、(d,x,y)(d, x, y)(d,x,y) を返します。

6. まとめ

拡張ユークリッドの互除法は、以下のように動作します：

  1. 基本ケース：
    b=0b = 0b=0 なら、gcd⁡(a,0)=a\gcd(a, 0)=agcd(a,0)=a として x=1,y=0x=1, y=0x=1,y=0 を返す。

  2. 再帰的に計算：
    a=bq+ra = bq + ra=bq+r として、再帰的に gcd⁡(b,r)\gcd(b, r)gcd(b,r) とその係数 x1,y1x_1, y_1x1​,y1​ を求める。

  3. 係数の更新：
    得られた x1,y1x_1, y_1x1​,y1​ を用いて、
    x=y1,y=x1−q y1x = y_1,\quad y = x_1 - q\,y_1x=y1​,y=x1​−qy1​
    として、ax+by=gcd⁡(a,b)ax + by = \gcd(a, b)ax+by=gcd(a,b) を満たす xxx と yyy を計算する。

この方法により、単に最大公約数を求めるだけでなく、aaa と bbb の線形結合としてその最大公約数を表現する整数解も得ることができます。
